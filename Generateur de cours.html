Présentation des Fonctionnalités

Cet outil web est conçu pour assister les enseignants de technologie au collège dans la création de supports pédagogiques variés et interactifs. Il se divise en plusieurs sections principales :

1. Génération de Prompts pour Modules de Cours (Section 1)
Permet de créer des instructions détaillées (prompts) à fournir à une Intelligence Artificielle (IA) pour la génération de modules de cours complets. L'enseignant sélectionne le niveau (5e, 4e, 3e), le module désiré, et peut personnaliser les thèmes à aborder. Le format de sortie du module (DOCX ou page web HTML) influence les instructions. Des outils annexes génèrent une phrase pour le cahier de texte et un résumé du module pour les élèves.

2. Génération de Prompts pour Exercices (Section 2)
À partir d'un contenu de cours, cette section génère des prompts pour créer divers types d'exercices avec une IA : Questions à Réponse Courte, QCM (demandant un format tableau à l'IA), Résumé de Cours, et Exercice de Regroupement, avec des options de personnalisation.

3. Conversion de Texte d'Exercices en Supports Visuels (Section 3)
Transforme le texte d'exercices en images ou en documents DOCX. Pour les Questions Courtes (format Question :: Réponse), génère une image ou un DOCX, avec ou sans correction. Pour les QCM, si l'IA fournit un tableau Markdown, un bouton le convertit en format texte standardisé. Ensuite, ce format peut être utilisé pour générer une image ou un DOCX, avec ou sans correction.

4. Génération d'Image Combinée des Exercices (Section 4)
Crée une image unique regroupant les questions courtes et les QCM préparés dans la section 3, avec option d'afficher ou masquer les corrections.

Objectif global : Faciliter et accélérer la création de matériel pédagogique en tirant parti des IA génératives, tout en offrant des outils de formatage et d'export pratiques.
