Plongez dans l'univers fascinant des réseaux qui façonnent notre monde numérique moderne.

Commencer l'apprentissage

Introduction : Le monde interconnecté

Qu'est-ce qu'un réseau informatique ?

Un réseau informatique est un ensemble d'appareils (ordinateurs, smartphones, imprimantes, etc.) connectés entre eux. Cette connexion permet le partage d'informations et de ressources entre les différents appareils.

Dans notre société moderne, les réseaux informatiques sont omniprésents et jouent un rôle fondamental dans :

Communication

Messageries instantanées, réseaux sociaux, e-mails, visioconférences

Information

Accès aux sites web, encyclopédies en ligne, actualités, recherches

Divertissement

Streaming vidéo, musique en ligne, jeux multijoueurs

Travail

Télétravail, partage de documents, collaboration à distance

Le Saviez-Vous ?

Le nombre d'appareils connectés à Internet dépasse aujourd'hui le nombre d'humains sur Terre ! On estime qu'en 2023, plus de 30 milliards d'appareils sont connectés au réseau mondial.

I. Les bases d'un Réseau

1.1. Les composantes essentielles

Les terminaux

Les terminaux sont les appareils utilisés par les utilisateurs finaux pour accéder au réseau. Ils peuvent être de différents types :

  • Ordinateurs fixes et portables
  • Smartphones et tablettes
  • Consoles de jeux connectées
  • Téléviseurs intelligents (Smart TV)
  • Objets connectés (IoT) : montres connectées, thermostats, etc.

Chaque terminal possède une interface réseau qui lui permet de se connecter.

Les moyens de transmission

Les moyens de transmission sont les supports physiques ou sans fil :

Câbles Ethernet (RJ45)

Connexion filaire courante, stable et rapide.

Fibre optique

Utilise la lumière pour très haut débit sur longues distances.

Ondes Wi-Fi

Connexion sans fil utilisant des ondes radio.

Satellite

Connexion dans les zones isolées.

Les équipements réseaux

Dispositifs pour l'interconnexion et la gestion du trafic :

  • Commutateurs (switches) : Connectent les appareils dans un LAN (basé sur adresse MAC).
  • Routeurs : Connectent différents réseaux et acheminent les données (vers Internet notamment).
  • Points d'accès Wi-Fi : Connexion sans fil au LAN.
  • Modems : Conversion numérique/analogique (lignes téléphoniques/câblées).
  • Pare-feu (firewalls) : Protègent contre les accès non autorisés.

Les serveurs

Ordinateurs spécialisés fournissant des services aux clients.

Types de serveurs :

  • Serveur web : Héberge des sites Internet (HTTP).
  • Serveur de fichiers : Stocke et partage des fichiers.
  • Serveur de messagerie : Gère les e-mails.
  • Serveur de base de données : Gère les données structurées.
  • Serveur DNS : Traduit les noms de domaine en adresses IP.
  • Serveur d'impression : Gère les imprimantes partagées.

1.2. Transmission des données

Les données sont divisées en petits paquets envoyés séparément.

Anatomie d'un paquet de données

  • Adresse IP source et destinataire
  • Données (partie du message)
  • Informations de contrôle (intégrité)
  • Numéro de séquence (ordre)

Le routage des paquets

Le routeur examine l'adresse de destination et détermine le meilleur chemin pour acheminer le paquet.

Le Saviez-Vous ?

La fibre optique transmet les données à environ 200 000 km/s.

II. Types de réseaux

Classification par taille et portée

2.1. Réseaux locaux (LAN)

LAN (Local Area Network) : Zone géographique restreinte (maison, école, bureau).

Caractéristiques : Hauts débits (100 Mbps - 10 Gbps), faible latence, administration unique.

Exemples : Réseau Wi-Fi domestique, réseau Ethernet d'entreprise.

2.2. Réseaux étendus (WAN)

WAN (Wide Area Network) : Zone géographique étendue (villes, pays), connecte plusieurs LAN.

Caractéristiques : Grandes distances, débits plus faibles, latence plus élevée, infrastructures publiques.

Exemples : Internet, réseau d'une multinationale.

Autres types

PAN (Personal Area Network) : Très courte portée (Bluetooth).

MAN (Metropolitan Area Network) : Échelle d'une ville.

CAN (Campus Area Network) : Campus universitaire ou complexe d'entreprise.

2.3. Internet : le réseau des réseaux

Internet est un système mondial d'interconnexion de millions de réseaux.

Structure d'Internet

  • Dorsales (Backbones) : Réseaux à très haut débit.
  • FAI (Fournisseurs d'Accès Internet) : Connectent les utilisateurs aux dorsales.
  • Points d'échange Internet (IXP) : Échange de trafic entre FAI.

Le World Wide Web (WWW)

Le Web est un service fonctionnant sur Internet, utilisant le protocole HTTP/HTTPS pour accéder à des ressources liées par des hyperliens.

Autres services sur Internet : e-mail, FTP, VoIP...

Le Saviez-Vous ?

Le premier message sur ARPANET (ancêtre d'Internet) en 1969 était "LO" (pour LOGIN), avant que le système ne plante.

III. Le Fonctionnement des Réseaux

3.1. Moyens de connexion

Câbles

Ethernet (RJ45)

  • Standard filaire courant
  • Stable, fiable
  • Mobilité réduite

Fibre optique

  • Très haut débit (lumière)
  • Longues distances
  • Coût d'installation

Sans fil (Wi-Fi)

Utilise des ondes radio.

Normes Wi-Fi

  • 802.11n (Wi-Fi 4)
  • 802.11ac (Wi-Fi 5)
  • 802.11ax (Wi-Fi 6)

Avantages

  • Mobilité, installation simple

Inconvénients

  • Sensible aux interférences, portée limitée, sécurité
Autres technologies sans fil
  • Bluetooth : Courte distance
  • Réseaux cellulaires : 4G/5G
  • ZigBee : Objets connectés

3.2. Protocoles de communication

Les protocoles sont des ensembles de règles pour la communication.

TCP/IP : la suite de protocoles d'Internet

ProtocoleFonctionCaractéristiques

Le Saviez-Vous ?

Le protocole HTTPS crypte les informations échangées (cadenas dans le navigateur).

Autres protocoles importants

  • FTP : Transfert de fichiers
  • SMTP : Envoi d'e-mails
  • POP3/IMAP : Réception d'e-mails
  • DHCP : Attribution automatique d'adresses IP
  • SSH : Connexion sécurisée à distance

IV. Adresses IP et Adressage

4.1. Adresses IP

Une adresse IP (Internet Protocol) est un identifiant numérique unique pour chaque appareil sur un réseau.

Adresses IPv4

  • Format : 4 nombres (0-255) séparés par des points (ex: 192.168.1.10)
  • Total : ~4,3 milliards

Adresses IPv6

  • Format : 8 groupes de 4 chiffres hexadécimaux (ex: 2001:db8::8a2e:370:7334)
  • Total : Immense (3,4 × 1038)
  • Conçu pour pallier le manque d'adresses IPv4.
Exemple concret

Le DNS traduit les noms de domaine (www.exemple.com) en adresses IP pour que votre ordinateur trouve le serveur du site.

4.2. Adresses MAC

L'adresse MAC (Media Access Control) est l'adresse physique fixe de l'interface réseau, attribuée par le fabricant.

Caractéristiques

  • Format : 6 groupes de 2 chiffres hexadécimaux (ex: 00:1A:2B:3C:4D:5E)
  • Utilisée pour la communication au sein d'un même réseau local (LAN).

Différence IP vs MAC

CaractéristiqueAdresse IPAdresse MAC
Niveau OSIRéseau (3)Liaison (2)
Attribuée parAdmin réseau/DHCPFabricant
Peut changerOuiNon (normalement)
UtilitéRoutage inter-réseauxCommunication intra-réseau

4.3. Le rôle du routeur

Le routeur sert d'intermédiaire entre le réseau local et les réseaux externes (Internet).

Fonctions principales

  • Attribution d'IP (DHCP) : Donne des adresses IP aux appareils.
  • Routage : Choisit le meilleur chemin pour les paquets.
  • NAT : Traduit IP privées en IP publique.
  • Pare-feu : Protège le réseau local.
  • QoS : Priorise certains trafics.

La box Internet

Combine souvent : Modem, Routeur, Point d'accès Wi-Fi, Switch.

Adresses IP Privées

Utilisées dans les réseaux locaux (non visibles sur Internet) :

  • 192.168.x.x
  • 172.16.x.x à 172.31.x.x
  • 10.x.x.x

V. Sécurité des Réseaux

5.1. Menaces

Virus et logiciels malveillants

Malware : programmes nuisibles.

  • Virus : Se reproduit en s'attachant à des fichiers.
  • Vers : Se propagent seuls sur le réseau.
  • Chevaux de Troie : Déguisés en logiciels légitimes.
  • Ransomware : Chiffre les données contre rançon.
  • Spyware : Collecte des informations à l'insu.
Intrusions et piratage

Accès non autorisé.

  • Attaques par force brute : Deviner les mots de passe.
  • Exploitation de vulnérabilités : Utilisation de failles logicielles.
  • Attaques DoS/DDoS : Surcharger un service.
  • Homme du milieu (MitM) : Interception des communications.
Phishing et ingénierie sociale

Exploitation de la psychologie humaine.

  • Phishing : E-mails/sites frauduleux pour voler des infos.
  • Ingénierie sociale : Manipulation psychologique.
Espionnage des données

Interception de données.

  • Sniffing : Capture du trafic réseau.
  • Keyloggers : Enregistrement des frappes clavier.

5.2. Mesures de sécurité

Pare-feu (Firewall)

Surveille et filtre le trafic réseau.

Antivirus

Détecte et supprime les malwares.

Mots de passe forts

Longs, complexes, uniques.

HTTPS

Chiffre les communications web.

Mises à jour régulières

Corrigent les vulnérabilités.

VPN

Crée une connexion chiffrée (tunnel).
