Comprendre le Système Binaire

Imaginez un monde avec seulement deux chiffres : 0 et 1. C'est le système binaire, la langue des ordinateurs !

Alors que nous utilisons 10 chiffres (0-9) dans notre système décimal, les ordinateurs fonctionnent avec ces deux états : 0 (pas de courant, faux) et 1 (courant présent, vrai). Chaque 0 ou 1 est appelé un bit.

Pourquoi le binaire ?

Les circuits électroniques des ordinateurs distinguent facilement ces deux états "allumé" ou "éteint", ce qui rend le binaire parfait pour représenter l'information numériquement.

Lire un nombre binaire

Comme en décimal où chaque position est une puissance de 10 (unités, dizaines, centaines...), en binaire, chaque position est une puissance de 2.

Exemple : 1011 (binaire)

  1         0         1         1
(1 * 2³) + (0 * 2²) + (1 * 2¹) + (1 * 2⁰)
(1 * 8)  + (0 * 4)  + (1 * 2)  + (1 * 1)
  8      +   0      +   2      +   1   = 11 (décimal)

Comment faire à la main ?

Binaire vers Décimal (Ex: 10110)

  1. Notez les positions (droite à gauche, début 0): 4 3 2 1 0
  2. Gardez les puissances de 2 où il y a un '1': $2^4$, $2^2$, $2^1$
  3. Calculez et additionnez : $16 + 4 + 2 = 22$

Décimal vers Binaire (Ex: 26)

  1. Divisez par 2, notez le reste.
  2. Continuez avec le quotient jusqu'à 0.
  3. Lisez les restes de bas en haut.
26 / 2 = 13 reste 0
13 / 2 = 6  reste 1
 6 / 2 = 3  reste 0
 3 / 2 = 1  reste 1
 1 / 2 = 0  reste 1
Résultat (lu de bas en haut) : 11010
Le Binaire Exemples

Le Binaire en Action

Le binaire est partout en informatique ! Voici quelques exemples concrets.

Stockage de Données

Tout ce qui est numérique est stocké en binaire : images (pixels), musique (échantillons sonores), textes (caractères), programmes (instructions)...

Tailles et Unités

Les capacités de stockage sont basées sur le binaire :

  • Octet (Byte): 8 bits (souvent 1 caractère)
  • Kilooctet (Ko): 1024 octets (un petit document)
  • Mégaoctet (Mo): 1024 Ko (une chanson MP3)
  • Gigaoctet (Go): 1024 Mo (un film HD)
  • Téraoctet (To): 1024 Go (des milliers de films)

Note : Parfois, les fabricants utilisent des puissances de 10 (kilo=1000), mais en informatique, les puissances de 2 ($2^{10}=1024$) sont plus précises.

Couleurs Numériques (RGB)

Les couleurs sur écran sont souvent codées sur 24 bits : 8 bits pour le Rouge, 8 pour le Vert, 8 pour le Bleu. Chaque couleur a 256 niveaux ($2^8$).

Rouge pur : 11111111 00000000 00000000 (255, 0, 0)
Vert pur  : 00000000 11111111 00000000 (0, 255, 0)
Blanc     : 11111111 11111111 11111111 (255, 255, 255)
Gris moyen: 10000000 10000000 10000000 (128, 128, 128)

Cela permet plus de 16 millions de couleurs ($256 \times 256 \times 256$) !

Le Code ASCII

Comment un ordinateur comprend-il les lettres et symboles ? Grâce à des normes comme l'ASCII (American Standard Code for Information Interchange).

L'ASCII attribue un numéro unique (de 0 à 127) à chaque caractère (lettres majuscules/minuscules, chiffres, ponctuation...). Ce numéro est ensuite stocké en binaire (sur 7 ou 8 bits).

Exemples Opérations

Opérations Logiques Binaires

Les ordinateurs manipulent les bits avec des opérations logiques fondamentales.

ET (AND)

Le résultat est 1 si et seulement si les deux bits sont 1.

0 ET 0 = 0 | 0 ET 1 = 0
1 ET 0 = 0 | 1 ET 1 = 1

Exemple : 1101 ET 1011 = 1001

OU (OR)

Le résultat est 1 si au moins un des deux bits est 1.

0 OU 0 = 0 | 0 OU 1 = 1
1 OU 0 = 1 | 1 OU 1 = 1

Exemple : 1101 OU 1011 = 1111

XOR (OU Exclusif)

Le résultat est 1 si les deux bits sont différents.

0 XOR 0 = 0 | 0 XOR 1 = 1
1 XOR 0 = 1 | 1 XOR 1 = 0

Exemple : 1101 XOR 1011 = 0110

À quoi ça sert ?

  • Masques de bits (sélectionner/modifier des parties d'un nombre).
  • Conditions logiques dans les programmes.
  • Calculs rapides, cryptographie simple.
