Cours Technologie : Le réseau Wifi

1. Fonctionnement détaillé d’un réseau Wifi

Le réseau Wifi (Wireless Fidelity) est une technologie de communication sans fil permettant aux appareils numériques de se connecter entre eux et à Internet sans câbles.

Cette connexion repose sur la transmission d’ondes électromagnétiques dans la bande des radiofréquences (2,4 GHz ou 5 GHz). Ces ondes sont émises par un routeur Wifi (ou point d’accès).

Le routeur est le chef d’orchestre :

  • Il diffuse le nom du réseau (SSID).
  • Il gère la sécurité (mot de passe WPA).
  • Il attribue une adresse IP locale à chaque appareil.
  • Il achemine les données entre les appareils et Internet.

Le système utilise des protocoles comme TCP/IP pour organiser l'échange de données en paquets.

Fonctions avancées du routeur : Filtrage, QoS, Gestion utilisateurs, Pare-feu.

2. Le rôle et le fonctionnement des adresses IP

L’adresse IP est un identifiant numérique essentiel pour l'acheminement des données.

Types d'Adresses IP : Publique vs Privée

  • IP Publique : Visible sur Internet, fournie par le FAI au routeur. (Ex: 82.65.145.203)
  • IP Privée : Utilisée dans le réseau local (LAN), attribuée par le routeur. (Ex: 192.168.1.2)

Exemple Concret de Fonctionnement (NAT)

Le routeur "traduit" les adresses privées en adresse publique pour communiquer avec l'extérieur et inversement.

  1. Appareil (IP privée) envoie requête au routeur.
  2. Routeur remplace IP privée par IP publique (NAT) et envoie sur Internet.
  3. Réponse revient à l'IP publique du routeur.
  4. Routeur (via NAT) retrouve l'appareil (IP privée) et lui transmet la réponse.

Plages d'Adresses IP Privées

  • Classe A : 10.x.x.x (Grandes entreprises)
  • Classe B : 172.16.x.x à 172.31.x.x (Moyennes entreprises)
  • Classe C : 192.168.x.x (Réseaux domestiques/petites entreprises)

IPv4 vs IPv6

IPv6 a été créé pour pallier le manque d'adresses IPv4 disponibles.

Masque de Sous-Réseau

Le masque de sous-réseau aide à déterminer si une destination est sur le même réseau local.

3. Cycle de vie d’un réseau Wifi

Les équipements Wifi (routeurs, cartes) ont un cycle de vie matériel :

  1. Conception & Développement : Ingénierie, normes (Wifi 6, 7...), tests.
  2. Fabrication : Production des composants, assemblage (souvent en Asie).
  3. Distribution & Vente : Transport, mise sur le marché.
  4. Installation & Configuration : Mise en place par l'utilisateur.
  5. Utilisation & Maintenance : Fonctionnement quotidien, mises à jour du firmware.
  6. Fin de Vie : Obsolescence, panne. Options : réutilisation, recyclage (DEEE), stockage, abandon.

Durée de vie typique d'un routeur : 3-5 ans (évolution des normes, arrêt des MàJ sécurité).

4. Les besoins énergétiques d’un réseau Wifi

Le Wifi dépend entièrement de l'énergie électrique.

Consommation du Routeur

Souvent allumé 24/7, un routeur consomme entre 6 et 12 Watts en moyenne.

  • Consommation annuelle : ~80-100 kWh (comparable à un petit appareil en veille).

Consommation des Appareils Connectés

Smartphones, tablettes, etc., consomment de l'énergie pour la connexion Wifi.

  • Le Wifi est généralement moins énergivore que les réseaux mobiles (4G/5G) sur un smartphone.

L'impact énergétique global inclut aussi la fabrication et les infrastructures réseau (data centers).

5. Les matériaux d’un réseau Wifi

Les équipements Wifi sont faits de matériaux variés, souvent non renouvelables.

  • Plastiques (Boîtier) : ABS, Polycarbonate (dérivés du pétrole).
  • Métaux : Cuivre (pistes PCB, fils, antennes), Aluminium (dissipateurs), Étain (soudure), Or (connecteurs).
  • Composants Électroniques : Silicium (puces), Céramique (résistances, condensateurs), Terres rares (parfois).

Le recyclage (DEEE) est important pour récupérer ces matériaux.

6. La chaîne d’énergie dans le fonctionnement du Wifi

Comment l'électricité de la prise devient des ondes Wifi :

  1. Source : Prise électrique (Courant Alternatif - AC, 230V).
  2. Conversion : Bloc d'alimentation transforme AC en Courant Continu (DC) basse tension (ex: 12V). Pertes par chaleur.
  3. Distribution : Le courant DC est distribué aux composants internes (processeur, puce Wifi...).
  4. Action finale : L'énergie alimente les puces et surtout les amplificateurs radio qui envoient le signal aux antennes. Celles-ci émettent les ondes électromagnétiques (Wifi).

Chaque étape génère de la chaleur (effet Joule).
